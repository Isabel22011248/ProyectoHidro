# Hidroeye 💧

Sistema inalámbrico de monitoreo de calidad del agua de riego usando LoRa y 4G.

## 🚀 Características
- Monitoreo de pH, turbidez y temperatura
- Comunicación LoRa
- Envío de datos por MQTT usando SIM7000
- Visualización en Node-RED y Grafana

## 🛠️ Tecnologías
- Arduino / ESP32
- LoRa
- MQTT
- InfluxDB
- Grafana

## 📊 Aplicación
Dirigido a agricultores del Valle del Mezquital para mejorar la calidad del agua de riego.
